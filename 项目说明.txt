【项目简介】

这是一个专业的币安现货山寨币筛选系统，专门用于在BTC/ETH开启
季度级别上涨时，自动筛选并抓取高Beta机会。

核心功能：
✓ 自动筛选币安所有现货山寨币
✓ 计算相对BTC/ETH的价格强度
✓ 多维度技术指标分析 (SMA, EMA, MACD, RSI等)
✓ 成交量分析和异动检测
✓ 自动生成K线图表，标注异动位置
✓ 邮件和Telegram实时推送
✓ 专业Web界面，操作简单
✓ 定时自动监控，无需人工值守

【技术架构】

后端:
- Python 3.8+
- FastAPI (高性能Web框架)
- SQLAlchemy (数据库ORM)
- CCXT (交易所API集成)
- Pandas + Pandas-TA (数据分析和技术指标)
- Plotly (交互式图表)

前端:
- React 18
- Ant Design (企业级UI组件)
- Axios (HTTP客户端)
- Recharts (数据可视化)

数据库:
- SQLite (轻量级，无需配置)

【快速开始】

第一次使用:
1. 运行 install.bat 安装依赖
2. 复制 .env.example 为 .env
3. 编辑 .env，填入币安API密钥
4. 运行 test_setup.py 测试环境
5. 运行 start.bat 启动服务

日常使用:
- 运行 start.bat，选择启动模式
- 浏览器访问 http://localhost:3000
- 设置参数，开始筛选

【核心功能详解】

1. Beta筛选
   - 计算山寨币/BTC和山寨币/ETH的价格比率
   - 筛选涨幅超过BTC/ETH的强势币种
   - 捕捉相对强度机会

2. 技术指标分析
   - 移动平均线: SMA(20,50,200), EMA(7,14,30,52)
   - 动量指标: MACD金叉检测
   - 超买超卖: RSI指标
   - 波动区间: 布林带
   - 趋势确认: 价格必须在主要均线之上

3. 成交量确认
   - 最小24h成交量过滤 (默认100万美元)
   - 成交量激增检测 (超过20日均量1.5倍)
   - 流动性保障

4. 价格异动检测
   - 实时监测5分钟、15分钟级别的价格突变
   - 自动标记异动位置
   - 捕捉早期机会

5. 智能评分系统
   - Beta分数 (30%): 相对强度
   - 成交量分数 (20%): 流动性
   - 技术分数 (50%): 综合技术指标
   - 总分0-100，推荐关注70分以上

6. 自动通知
   - 邮件推送 (支持Gmail, QQ邮箱等)
   - Telegram机器人推送
   - 实时提醒高分机会
   - 可自定义推送阈值

7. 可视化图表
   - 自动生成专业K线图
   - 标注技术指标 (SMA, EMA, 布林带)
   - 标记价格异动点
   - 支持多时间周期

8. 定时监控
   - 可设置监控间隔 (默认5分钟)
   - 自动筛选所有币种
   - 发现机会自动通知
   - 无需人工值守

【使用建议】

1. 市场时机
   - BTC/ETH处于明显上升趋势时效果最好
   - 震荡或下跌市场建议谨慎使用

2. 参数设置
   - 保守: 1h周期, 500万成交量, 75分以上
   - 激进: 5m周期, 100万成交量, 65分以上
   - 稳健: 15m周期, 200万成交量, 70分以上 (推荐)

3. 风险管理
   - 不要全仓单一币种
   - 建议分散3-5个标的
   - 设置合理止损 (5-8%)
   - 注意仓位管理

4. 确认信号
   - 多时间周期共振
   - 成交量配合放大
   - 技术指标多重确认
   - 结合大盘走势
